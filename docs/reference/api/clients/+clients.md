# Clients

This section contains documentation for client-side applications and integrations with the Gaia Platform API.

---

## Client Documentation

**[client-usage-guide.md](client-usage-guide.md)**
*   **Summary**: This guide explains how client applications should interact with the Gaia Platform API, emphasizing that all requests must go through the single gateway endpoint. It details the supported authentication methods (API Key, JWT), provides examples for key API endpoints like asset management and chat, and includes client SDK examples in JavaScript, C#, and Python.

**[GAIA_CLI_CLIENT.md](GAIA_CLI_CLIENT.md)**
*   **Summary**: This document provides instructions for setting up and using the interactive command-line interface for the GAIA platform. It covers installation in a Python virtual environment, secure API key configuration using the OS keyring, and usage for both interactive chat and batch-mode scripting. It also details the available interactive commands for managing conversations and personas.
