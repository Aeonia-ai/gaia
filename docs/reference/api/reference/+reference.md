# Reference

This section provides detailed technical references for the Gaia Platform, including API specifications and architectural maps.

---

## API Reference Documents

**[CLIENT_API_REFERENCE.md](CLIENT_API_REFERENCE.md)**
*   **Summary**: This document is a client-focused API reference for external developers, recommending the use of the clean v0.3 API. It details public, authentication, chat, and conversation management endpoints, providing `curl` examples and client SDK code in Python and JavaScript. It also includes error handling and rate-limiting information.

**[GAIA_API_MAP.md](GAIA_API_MAP.md)**
*   **Summary**: This document provides a comprehensive mapping of all API endpoints through the GAIA platform services. It details how the Gateway routes requests to backend services like Auth, Chat, Asset, and KB, and lists the specific paths for each service, covering API versions v0.2, v0.3, and v1.

**[GAIA_API_REFERENCE.md](GAIA_API_REFERENCE.md)**
*   **Summary**: This document serves as an advanced API reference for platform developers, covering the full feature set including personas and asset management. It provides details on both the clean v0.3 API and the OpenAI-compatible v1 API, with examples for chat, conversation management, AR/location endpoints, authentication, and asset management.
